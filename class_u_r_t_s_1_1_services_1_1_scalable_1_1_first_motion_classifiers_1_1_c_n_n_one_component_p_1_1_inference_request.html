◆ setThreshold()

void URTS::Services::Scalable::FirstMotionClassifiers::CNNOneComponentP::InferenceRequest::setThreshold ( double  threshold)

Defines the probability threshold above which an up or down probability must exceed to actually be classified as up or down.

Parameters
[in]thresholdThe probability threshold. If an up or down first motion is larger than the unknown probability and exceeds this value then the first motion will be assigned to whichever is larger; probability up or probability down. Otherwise, the probability will be unknown.
Exceptions
std::invalid_argumentif the threshold is not in the range [0,1].
