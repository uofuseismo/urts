Detailed Description

Defines an origin which is an event location built from arrivals.

enum URTS::Broadcasts::Internal::Origin::Origin::ReviewStatus : int8_t

Sets the review status as either being purely automatic - i.e., there was no review, or the pick was manually reviewed, possibly edited, then published.

Enumerator

This is an automatically generated origin.

Manual 

This is origin was created or reviewed by a human.
