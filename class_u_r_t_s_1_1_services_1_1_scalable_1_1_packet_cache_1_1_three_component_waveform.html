Detailed Description

This class converts a sequence of packets for a three-component station to three continuous waveforms that have the same start time, length, and sampling rate.

Sets the data response and interpolates to a continuous signal.

Parameters
[in]verticalComponentThe data response from the packet cache for the vertical component.
[in]northComponentThe data response from the packet cache for the north (or 1) component.
[in]eastComponentThe data response from the packet cache for the east (or 2) component.
Note
This will interpolate the packets to the nominal sampling rate and set the corresponding channel naming information.
Exceptions
std::invalid_argumentif the naming information cannot be determined, there are no packets, or the packets correspond to different channels.
std::runtime_errorif an error occurs during interpolation.

◆ setGapTolerance()

void URTS::Services::Scalable::PacketCache::ThreeComponentWaveform::setGapTolerance ( const std::chrono::microseconds &  gapTolerance)
noexcept

Sets the gap tolerance between packets.

Parameters
[in]gapToleranceIf the time between the end of a packet and the start of the subsequent packet exceeds this time then the samples interpolated between the packets will be denoted as extrapolated data gaps.
Note
Zero or negative effectively disables this. In which case any sample interpolated between a packet is being extrapolated in a gap.
