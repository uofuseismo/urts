enum URTS::Services::Scalable::PacketCache::DataResponse::ReturnCode

Defines the return code for a data request.

Enumerator
Success 

No errors were detected; the request was successful.

NoSensor 

The data for the requested sensor (Network, Station, Channel, Location Code) does not exist.

InvalidMessageType 

The received message type is not supported.

InvalidMessage 

The request message could not be parsed.

InvalidTimeQuery 

The time query parameters are invalid.

AlgorithmicFailure 

An internal error was detected .
