void URTS::Broadcasts::External::SEEDLink::StreamSelector::setSelector ( const std::string &  channel,

)

Sets the desired channels.

Parameters
[in]channelThe channel selector. For example, this can be BH? to read all BH[1,2,Z] channels. Or this can be HHZ to read only high-samplerate, high-gain vertical channels. Or, if this is blank then all channels will be selected. Or, this can be !L to exclude LCQ and LEP channels.
[in]locationCodeThe location code. For example, this can be 01 to return only data with 01 location codes. If this is empty then all location codes will be selected.
[in]typeThe data type to be returned.
