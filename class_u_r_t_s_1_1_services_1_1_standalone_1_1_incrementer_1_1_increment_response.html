◆ ReturnCode

enum URTS::Services::Standalone::Incrementer::IncrementResponse::ReturnCode

Defines the return code for an increment request.

Enumerator
Success 

No errors were detected; the request was succesful.

NoItem 

The desired item could not be found.

InvalidMessage 

The message could not be parsed.

AlgorithmFailure 

An internal counting error was detected. The returned value should not be trusted to be unique.
